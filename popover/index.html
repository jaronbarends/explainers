Basic popover - auto state

basic popover

basic auto popover content
2nd basic auto popover content

Simplest example just needs an element with a popover attribute (or popover="auto") and an id, and a button with a popovertarget attribute referencing the id. The button toggles the popover. 

<div id="popover-basic-auto" popover>
  basic auto popover content
</div>
<button popovertarget="popover-basic-auto">
  toggle popover
</button>

basic popover - auto state with action

basic auto action popover content

Adding a popovertargetaction attribute of show, hide or toggle. Without popovertargetaction, behavior defaults to toggle.

auto popover behavior

Popovers with auto state automatically close other auto popovers; they can be dismissed with esc or by clicking outside of the popover.

Basic popover - manual

basic popover - manual

basic manual popover content

With popover="manual", you can control the popover with popovertargetaction just like with the auto popovers, but you can also call the showPopover, hidePopover and togglePopover methods on the popover element.

Multiple manual popovers

With popover="manual" you can have multiple popovers open at the same time

Styling popovers

Basic styling example
You can style the popover just like any normal element. As an extra, it has the :popover-open pseudo-class, and the ::backdrop pseudo-element.
