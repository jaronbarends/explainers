Simple masks

used throughout this page: --near-transparent: rgba(0, 0, 0, 0.03);

content
no mask
content
mask: linear-gradient( black, transparent );

mask

  • Everything beneath the transparent part gets masked
  • Everything beneath opaque part is shown

content
mask: linear-gradient(
  black,
  black 25%,
  var(--near-transparent) 0,
  var(--near-transparent) 50%,
  white 0,
  white 75%,
  var(--near-transparent) 0
);

mask

  • Everything beneath the transparent part gets masked
  • Everything beneath opaque part is shown - mask color does not matter

Multiple masks

Masks used:

content
--mask-rectangle-landscape: linear-gradient(
  var(--near-transparent),
  var(--near-transparent) 40%,
  black 0,
  black 60%,
  var(--near-transparent) 0
);
mask: var(--mask-rectangle-landscape);

content
--mask-circle: radial-gradient(
  black,
  black 50%,
  var(--near-transparent) 0
);
mask:var(--mask-circle);

content
--mask-rectangle-portrait: linear-gradient(
  to right,
  var(--near-transparent),
  var(--near-transparent) 70%,
  black 0,
  black 90%,
  var(--near-transparent) 0
);
mask: var(--mask-rectangle-portrait);
