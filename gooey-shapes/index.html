When contrast value is too low, you get ugly anti-aliasing at the overlap 

.blob-container {
	filter: contrast(10);
	background: white;
}

.blob {
	filter: blur(0.1 * var(--blob-size));
}

When contrast value becomes to high, edges get pixelated 

.blob-container {
	filter: contrast(50);
	background: white;
}

.blob {
	filter: blur(0.1 * var(--blob-size));
}

When blur-radius becomes too big, contrast turns pixels near the outer edge to white. When radius is really big, this may even turn entire blob white. Use a blur-radus that is depending on the blob's size. 

.blob-container {
	filter: contrast(25);
	background: white;
}

.blob {
	filter: blur(0.4 * var(--blob-size));
}

When using colors, contrast changes the color values too. It only works correctly when rgb channel values of 0 and 255. Using different colors is possible with svg-filters (not part of this explainer). 

.blob {
	background: purple;
}

You can use any color you want on a black background using mix-blend-mode 

.blob-container {
	filter: contrast(25);
	background: white;
}

.blob {
	filter: blur(0.1 * var(--blob-size));
}
